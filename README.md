## **Shopping App TypeScript**

### *Project Description:*
Shopping App TypeScript is a React project and it was created using create-react-app. To style the project it was used Tailwind CSS.

### *Getting started:* 
  - First install all the dependencies:
  ```
  npm install
  ```
  - Then to run it locally:
  ```
  npm run dev
  ```
  - In case that you want to see a live demo, you can find it available in this link:  
    [Shopping App TypeScript](https://shopping-app-typescript-js6j.vercel.app)
